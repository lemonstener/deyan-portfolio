Some of the things I have worked on

  • Commercio Orders

    Tech stack: React, TypeScript, Material UI, Recoil/Redux, React Testing Library with Jest, C#, SQL, Azure DevOps, REST, AGILE

    Commercio Orders is a CRM platform for Promotional Products Industry distributors. Users are able to source competitively-priced products, manage quotes, and send purchase orders with less back and forth with the suppliers, all in one system.

    I have worked on almost every feature of this application, but most notably I have:

    • Created internal tools and components for form handling which decreased development time of forms by 90%, significantly improved form performance across the application, reduced unnecessary rerenders, eliminated huge chunks of unoptimized code, reduced dependencies, and provided a consistent UX across forms
    • Increased test coverage by 22% on the front end
    • Fully owned the Status Management feature for Purchase Orders, Quotes, and Products on the front end and did additional work on the back end
    • Fully owned the Supplier Connection feature on the front end - implemented the landing page, forms, credentials management, UI components and more
    • Fully owned the User Management feature on the front end - implemented the landing page, user table, forms, invitations, admin controls, and more
    • Worked on the Purchase Orders Overview feature as full-stack developer, creating the necessary web API endpoints on the back-end, and then integrating them on the front end. Other work includes querying, sorting, filters, UI, and more
    • Implemented the Digital Art Proof feature for purchase orders on both the front and back end
    • Led the charge on the complete refactor of the Purchase Order Configuration and Sharing features on the front end, which eliminated numerous bugs and typescript errors, improved performance, organized and simplified the codebase, and eliminated over 4000 lines of inefficient code

    Other duties included writing documentation, code reviews, participating in SCRUM ceremonies, and more.
